Definition : Locally Lipschitz Continuous

We say f:Rn→Rm is Locally Lipschitz continuous at x, such that

If∃ρ>0&L>0then∀y∈Bo(x,ρ),∥f(x)−f(y)∥≤L⋅∥x−y∥

  • x 근방의 미분치는 L 보다 작다는 의미이며 꼭 미분 가능이 아니라도 성립한다.

Convex Sets

We say a set A⊂Rn is convex,
if every x,y∈A, the line connexting them is also in A. i.e.

∀x,y∈A,x+λ(y−x)∈A,∀λ∈[0,1]

Definition : Convex Hull

Let S⊂Rn, A convex hull, coS, is the smallest convex set containing the set S.
