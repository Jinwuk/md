Norm

A Norm in Rn is a function

||⋅||:Rn→R+

  1. ||x||=0 iff x=0
  2. ||λx||=|λ|||x||∀λ∈R,x∈Rn
  3. ||x+y||≤||x||+||y||∀x,y∈Rn

Open Set and Closed Set

Definition 1. Open ball

∀x∈Rn and ρ>0, by

Bo(x,ρ)={z∈Rn|||z−x||<ρ}

we denote an open ball with radius ρ.


Figure 1. Open Ball

Definition 2. Open Set

A Set Z⊂Rn is said to be a Open,

if∀x∈Zand∃ρ>0,∃Bo(x,ρ)⊂Z


Figure 2. Open Set

Example

An open ball Bo(x,ρ) is am open set.

Definition 3. Closed Set

A set Z⊂Rn is said to be Closed (Set) , if its complement in Rn is open.

  • 즉, Zc 가 Open Set 이면 Z 는 Close Set.

Definition 4. Compact

A set Z⊂Rn is said to be Compact , if every open covering of Z has a finite open covering

Note

A Compact set in Rn is closed and bounded.

  • 모든 원소의 Open subset 으로 Infinite 하게 Cover되는 Open Set의 여집합이면 Closed Set 이면서 Finite하게 Cover 된다.

Example

Is a set (0,1] closed?

Set ui=(0,1−1i]. The set (0,1] is OPEN since ⋃∞i=0ui⊃(0,1] through ⋃∞i=0ui⊇(0,1−ϵ].

  • Finite가 아니므로 Open

Sequence

A Sequence is a function from N to Rn such that

{xi}∞i=1

A Subsequence of {xi}∞i=1 is a set {xi}i∈K where K⊂N has an infinite number of elements.

Limit Point and Accumulation points

Definitioon 5. Limit Point (Important!)

A sequence {xi}i∈N is said to be converge to x^,

iflimi→∞∥xi−x^∥=0.

or, (다음 정의가 더 중요하다.)

∀ϵ>0and∃i0∈N,∃∥xi−x^∥<ϵ,∀i≥i0

The point x^ is called a limit point.

Definitioon 6. Accumulation point

A point x∗ is called an accumulation point if,
There exists an infinite subsequence , {xi}i∈K such that xi→Kx∗
즉,

limi∈Ki→∞∥xi−x∗∥=0

  • 처음부터 시작하여 무한하면 Limit Point, 임의의 시작점에서 무한하면 Accumulation point

Theorem 1.

Z⊂R is closed iff a sequence {xi}i∈N⊂Z converges to x^ (i.e. xi→x^) then x^∈Z.

proof

Sufficient

Suppose that x^∉Z⇒x^∈Zc.
Since ∃ρ>0 such that Bo(x^,ρ)⊂Zc, and, by assumption, xi→x^forρ>0, ∃i0∈N, such that |xi−x^|<ρ,∀i≥i0. i.e. xi∈Bo(x^,ρ)⊂Zc,∀i≥i0 (Open Set의 정의에 따라 이는 당연, 즉, xi∈Zc).

However, by assumption, xi∈Z. 따라서 이는 가정에 모순이다.

Necessity

Suppose that Z is not close (i.e. open), it means that Zc is not open (i.e. close).
,and ∃x^∈Zc (결론이 반대라고 가정하면..) such that ∀ρ>0,Bo(x^,ρ)⊈Zc (i.e. Bo(x^,ρ)∩ZC=∅). (가정과 결론이 반대인 경우가 성립한다고 가정하면)
Let ρi=1i, and pick xi∈Bo(x^,ρi)∩Z, then xi→x^,xi∈Z,∀i. However, x^∈Zc, and it is contradict to the assumption.

[Q.E.D]

Continuity and Uniform Continuity

We say a function f:Rn→Rm is continuous at x∈Rn

If∀δ>0,∃ε>0such that∥f(y)−f(x)∥<δ∀y∈Bo(x,ε)

(Locally Continuouis)

We say a function is uniform continuous

If∀δ>0,∃ε>0such that∀x∈Rn,∥f(y)−f(x)∥<δ∀y∈Bo(x,ε)

(∀x∈Rn 조건에 의해 매우 강력한 Continuous 가 된다, 전체 영역에 대한 것이므로..)

HW 2.

f is continuous at x^ iff for any convergent sequence {xi}i∈N to x^, f(xi) also converges to f(x^).

Example

f:X→M is continuous, and X is compact then f is uniform continuous on X.

Proof

Since f is continuous, ∀x∈X,∀δ>0,∃εx>0 such that ∥f(y)−f(x)∥<δ implies that y∈Bo(x,εx).
Since X is compact, {Bo(x,εx)}x∈X covers X (i.e. ⋃x∈XBo(x,εx)⊃X)
Moreover, Since X is compact, there exists a finite number of open covering, say {xi}ki=1 such that ⋃ki=1Bo(xi,εx)⊃X.
Then, pick ε such that ε=mini={1⋯k}εxi. It implies that ε satisfies the property.

Lemma : Continuous on Compact Set

X⊂Rn is compact and f:Rn→Rm is continuous. It implies that f(X)={y∈Rm|y=f(x),x∈X}⊂Rm is compact.
(직관적으로 당연하나, 이것의 증명을 위해서는 Contionus 와 Compact의 정의를 정확히 사용하여야 한다.)

Proof

  • Rn is compact means that it is closed and bounded.
  • Proof of Closed : Limit Point가 같은 Set에 있음을 보인다.
    It needs to prove that for any sequence {yi}i∈N⊂f(Z) converges to y^∈f(X).

Since {yi}⊂f(X) for any yi∈f(X),∃xi∈X such that yi=f(xi).
Since X is compact, there exists subsequence of {xi}i∈N such that xi→Kx^ and x^∈X.
Since f is continuous xi→X^ implies that f(xi)→Kf(x^)
… 다시말해, x^ is a limit point, since if it is not {yi}, it cannot be a convergence sequence.
then f(xi)→Kf(x^)⇔yi→y^.

  • Proof of Bounded : 제한된 값을 가짐을 보인다.
    Suppose not, i.e. ∃{yi}i∈N such that yi↑∞ as i→∞.
    Since yi∈f(x),∃xi∈X such that xi→x^⇔f(xi)→f(x^)⇔yi→y^, thus, ∃i0∈N such that ∥f(xi)∥≤∥f(x^)∥+1<∞,∀i≥i0, It is contradict to assumption. Compact 가정에 어긋난다.

Taylor’s Formula

Suppose that f:Rn→Rm is continously differentiable (계속 미분 할 수 있다는 의미), then

∃y,x∈Rn,f(y)−f(x)=∫10∂f(x+s(y−x))∂x⋅(y−x)ds

Proof

Let g(s)≜f(x+s(y−x)) such that g(1)=y, and g(0)=x. By assumption, g(s) is differentiable since f is continuouslu differntiable.

dg(s)ds=df(x+s(y−x))dx⋅(y−x)

(by Chain rule, dg(s)ds=df(X(s))dX(s)⋅dX(s)ds,X(s)=x+s(y−x), Set X(s) to be x)

∫10dg(s)ds⋅ds⇔g(1)−g(0)=∫10∂f(x+s(y−x))∂x⋅(y−x)ds=∫10∂f(x+s(y−x))∂x⋅(y−x)ds
