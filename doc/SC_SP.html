  • 동시에 뭔가 새로운 것을 만들어 낼 수도 있다.

Definition

A stochastic process is a collection of random variables , each random variable {Xt} is on for t=1,2,...,T.

A stochastic process adapted to Filtration

  • When a r.v. Xt is Ft-measurable i.e. if ∀t∈[0,T],Xt is a random vriable on Ft, we say Xt is adapted to the Filtration F
  • 즉, 시간에 따라 확장되는 Ft를 정의역으로 하는 확률변수의 모임을 의미한다.

Definition : Gaussian Process

A stochastic process Xt,t≥0 is called a Gaussian Process, if Xt1,⋯Xtn has multivariate normal distribution for all t1⋯tn.

  • 위에서 정의한 Wt의 경우 De-Moivre-Laplace Theorem (1738)에 의해 t가 충분히 클 때, 혹은 Δτ가 충분히 작을 떄, 이항분포의 근사분포는 정규분포를 따른다.
  • 다시말해, t=tk−tk−1 가 많은 Δτ로 만들어질 수 있으면 Transition Probability p(t,x,y)는 Gaussin 혹은 정규분포를 따른다.

Definition : Wiener Process

A wtochastic process Wt∈mathbbR defined for t∈[0,∞] such that

  • W0=0 a.s.
  • the sample paths t↦Wt are a.s. continuous
  • for any finite sequence of times 0<t1<⋯<tn and Borel sets A1,⋯An⊂R,

    P(Wt∈A1,⋯,Wtn∈An)=∫A1⋯∫Anp(t1,0,x1)p(t2−t1,x1,x2)⋯p(tn−tn−1,xn−1,xn)dx1⋯dxn

    where p(t,x,y)=12πt√exp(−(x−y)22t) for x,y∈R and t>0. It is called a Transition Probability.

Wiener Process의 특성

  • 초기시간 0 , 위치 0 에서 시간 t에서 x 까지의 전이 확률은 fWt(x)=p(t,0,x)=12πt√exp(−x22t) 이것은 다음 2차 편미분 방정식의 해이다.

    ∂f∂t=12∂2f∂x2

    이는 Wiener Process의 해석에 미분을 사용할 수 있다는 의미이다.
  • Ws,Wt의 Covariance는 E(Wt,Ws)=min(t,s).
  • E(|Wt−Ws|2)=|t−s|. 이는 Independent Increment 라는 강력한 개념과 연결된다. (Linear Algebra와 연결된다.)
  • 전이 확률은 독립사건의 확률 분포함수이다 고로 fWs,Wt(x,y)=p(s,0,x)p(t−s,x,y) 따라서

    P(Wt−Ws∈A)=∫Ap(t−s,0,u)du

Martingale

A Stochastic Process {Xt,t≥0} adapted to a filtration F is a martingale if for any t,

  • {X(t),t≥0} is integrable, i.e. |E(Xt)|<∞
  • For any s<t (Core Definition)

    E(Xt|Fs)=Xs

  • Super Martingale E(Xt|Fs)≤Xs , Sub Martingale E(Xt|Fs)≥Xs

Doob-Levy Martingale

Let Y be an integrable r.v. , that is, E(|Y|)<∞, then Mt=E(Y|Ft) is a martingale.

E(Mt|Fs)=E(E(Y|Ft)|Fs)=E(Y|Fs)=Ms

  • 만일, 시간 t까지의 Filtration 의 한 σ-algebra Ft 을 그떄까지 정보의 집합 혹은 입력 데이터의 집합이라고 하면 이것을 통해 미래의 데이터 값을 유추할 수 있다는 의미가 된다.
  • Winer Process는 Martingale 이다. i.e. E(Wt|Fs)=Ws, for s<t.

Martingale의 특징

  • {exp(Wt−12t)} 는 Martingale 이다. i.e. E(exp(Wt−12t)|Fs)=exp(Ws−12s)
