Preliminaries

Continuity and Uniform Continuity

We say a function f:Rn→Rm is continuous at x∈Rn

If∀δ>0,∃ε>0such that∥f(y)−f(x)∥<δ∀y∈Bo(x,ε)

(Locally Continuouis)

We say a function is uniform continuous

If∀δ>0,∃ε>0such that∀x∈Rn,∥f(y)−f(x)∥<δ∀y∈Bo(x,ε)

(∀x∈Rn 조건에 의해 매우 강력한 Continuous 가 된다, 전체 영역에 대한 것이므로..)

Taylor’s Formula (or Differentiable)

Suppose that f:Rn→Rm is continously differentiable (계속 미분 할 수 있다는 의미), then

∃y,x∈Rn,f(y)−f(x)=∫10∂f(x+s(y−x))∂x⋅(y−x)ds

Twice Differentiable

Let f:Rn→R be a twice differentiable, then, for any x,y∈Rn

f(y)−f(x)=⟨∂f(x)∂x,y−x⟩+∫10(1−s)⟨y−x,∂2f(x+s(y−x))∂x2⋅(y−x)⟩ds

First and Second Order Necessary Condition

First Order Necessary Condition

Let Ω⊃Rn,andf∈C1,Ω⇒R.
If x∗ is a local minimum point of f over Ω, then ∀d∈Rn that is a feasible direction at x∗, we have

⟨∇f(x∗),d⟩≥0

  • 간단히 말하면 ∇f(x∗)=0, 이것이 First Order Necessary Condition

Second Order Necessary Condition

Suppose that f is twice continuosly differentiable. Let x^ is a local minimizer of minx∈Bo(x^,ρ)f(x).
Let H=∂2f∂x2 then ⟨h,H(x^)h⟩≥0,∀h∈Rn.

즉, x∗ is Local Minimum 이면

First order Necessary conditions Second order Necessary conditions
⟨∇f(x∗),h⟩≥0,h: Feasible  ⟨h,H(x∗)h⟩≥0,h: Feasible 

Theorem F-3 (Sufficient Condition)

Suppose that f is twice differential continously differentiable, and that

x^∈Rnsuch that∇f(x^)=0andH(x^)>0

Then x^ is a local minimizer of f.

Convex Function

Definition : Convex function

f:Rn→R is said to be convex . If ∀x,y∈Rn,λ∈[0,1], then

f(x+λ(y−x))≤(1−λ)f(x)+λf(y)

Theorem : Differentiable and Convex

f is differentiable then f is convex iff ∀x,y∈Rn

f(y)−f(x)≥⟨∇f(x),(y−x)⟩

Theorem : Twice Continuosly Differentiable and Convex

f is twice continuosly differentiable, then f is convex iff

∂2f(x)∂x2≥0,∀x∈Rn

Gradient Method

Corollary : descent direction

If x∈Rn is such that ∇f(x)≠0, then any vector h∈Rn such that ⟨∇f(x),h⟩<0 is a descent direction for f(⋅) at x. i.e.

∃λ^>0such thatf(x+λ^h)−f(x)<0

Directional Derivative

Directional Derivative at x along the vector h is defined by

df(x,h)=limλ↓0f(x+λh)−f(x)λ=⟨∇f(x),h⟩

where λ↓0 means that λ is Monotone decreasing on R+, and λ↑0 is Monotone increasing on R−.

  • If f is continuosly differentiable, then

f(x+λh)−f(x)=∫10⟨∇f(x+sλh),λh⟩⟹f(x+λh)−f(x)λ=∫10⟨∇f(x+sλh),h⟩ds

Let λ↓0, then

df(x,h)=limλ↓0f(x+λh)−f(x)λ=limλ↓0∫10⟨∇f(x+sλh),h⟩ds=∫10⟨limλ↓0∇f(x+sλh),h⟩ds=⟨∇f(x),h⟩∫10ds=⟨∇f(x),h⟩

Note

많은 경우 x를 생략하고 df(h) 를 많이 사용한다. 여기서는 최적화 알고리즘 도출을 위해서 x를 굳이 표시하였다. 그 외에 h[f]=h(f) 역시 같은 의미로 사용한다. (미분 기하학, 리만 기하학, 미분 다양체론 등에서..)

Newton’s Method (Local case)

Assumption

  • f is twice differentiable and locally Lipschitz, i.e. for given any bounded set S⊂Rn

    ∃L<∞such that∀x′,x∈S,∥f(x′)−f(x)∥≤L∥x′−x∥

  • ∃0<m≤M<∞ such that ∀y∈Rn

    m∥y∥2≤⟨y,H(x^)y⟩≤M∥y∥2

    Basic Idea

    For given f(⋅), we can approximate f near xi using a Taylor expansion such that

    f(x)≈f(xi)+⟨∇f(xi),x−xi⟩+12⟨x−xi,H(xi)(x−xi)⟩

    Local minimizer x에서 ∇f(x)=0 이므로 위 식을 x에 대하여 미분하면

    ∇f(x)=∇f(xi)+H(xi)(x−xi)=0⇒x−xi=−H−1(xi)∇f(xi)

    Procesdure Processing
    Data xo∈Rn
    Step 0 Set i=0
    Step 1 Compute the Search Direction
    hi=−∇f(xi)
    Stop If ∇f(xi)=0
    Step 2 Compute the Step Size
    * λi=λ(xi)=argminλ≥0f(xi+λhi)
    Step 3 Update xi+1=xi+λihi replace i by i+1 and goto Step 1

  • λi를 Armijo 방식에 의해 선정하면 Armijo Newton

Quadratic Problem and Conjugate Descent

Consider the Following problem

minf(x):f(x)=12⟨x,Qx⟩−bTx,Minimizerx∗=Q−1b

Let ∇f(x)=Qx−b=g(x)
Steepest Descent Method를 생각해 보면

xi+1=xi−αigi,whereαi=argminλ≥0f(xi−λgi)⟹∂f∂λ=0

0=∂f∂λ=∂f∂x∂x∂λ=(Q(x−λgi)−b)T(−gi)⟹λgTiQgi=(Qx−b)Tgi⟹λ=gTigigiQgi

위 λ를 만족하면 현재 Gradient와 다음 Gradient가 Orthogonal 하다는 의미가 된다. 이런 특성을 사용하여 만든 최적화 알고리즘이 Conjugate Gradient

General Conjugate Gradient

Quadratic 문제가 아니어도 Conjugate Algorithm을 적용하고 싶다. 그런데 Hessian이 Constant가 아닌 경우, Orthogonal한 Gradient를 어떻게 보장할 것인가?

  • 일반적인 Quadratic 문제에서는 다음과 같이 ri를 선정한다

    ri=−⟨Qhi,gi+1⟩⟨hi,Qhi⟩

    이렇게 되면 ⟨hi+1,Qhi⟩=0 이 되고 이렇게 되어야 {gk} 가 Orthogonal해진다. (증명 생략)

  • ri를 잘 선정하여 최대한 Orthogonal 하게 만들어 본다.

  • Polak-Riebel : Gradient 끼리 조금 Orthogonal이 아니라면 해당 부분 만큼 보정 한다.

    ri=⟨∇f(xi+1)−∇f(xi),∇f(xi+1)⟩∥∇f(xi)∥2

  • Fletcher-Reeves : Gradient는 다 Orthogonal 하다고 본다.

    ri=∥∇f(xi+1)∥2∥∇f(xi)∥2

Quasi Newton

원래 Newton Method는 xi+1=xi−λiH−1∇f(xi) 의 형태이다. 그런데, 최적화 문제가 언제나 Quadratic 하게 주어지는 것도 아니므로 Hessian이 고정적으로 나타난다는 보장도 없다. 따라서 Hessian을 최대한 Estimation하여 Newton Method와 비슷하게 만들고자 하는 것이 Quasi Newton이다.

Idea

Problem minx∈Rnf(x) 에서 f(x)를 f(x)=⟨d,x⟩+12⟨x,Hx⟩ 로 근사화 하고 Search Direction을 다음에서 구한다.

h(x)=argmin{12∥h∥2H+df(x,h)}

  • H=I 인 경우 Steepest Descent와 똑같다.

    ∂h(x)∂h=∂∂h(12hTh+∇f(x)Th)=h+∇f(x)=0⟹h=−∇f(x)

  • H가 Hessian이면 Newton Method와 같다.

    ∂h(x)∂h=∂∂h(12hTHh+∇f(x)Th)=Hh+∇f(x)=0⟹h=−H−1∇f(x)

  • Hessian을 근사화 하기 위해 B 라는 것을 놓자. 위의 Newton Method를 보면 Δxi=xi+1−xi, gi=∇f(xi)=Hxi+d 에서

    Δgi=gi+1−gi=Hxi+1+d−Hxi−d=HΔxi∴H−1Δgi=Δx

    여기서 BΔgi=Δxi,∀i 라고 생각하고 Δi 가 Nonsingular 이면 B=H−1=ΔXΔG−1 로 생각한다. (ΔX는 Δxi를 Column vector로 놓은 Matrix, ΔG는 Δgi를 Column Vector로 놓은 Matrix)
