## VCA SW 버전과 Server ROM BIOS
Dell의 경우 VCA SW 버전이 1.2 일 경우, HOST ROM BIOS의 Upgrade가 필요하다.
그러나 1.3.47 이후에는 필요 없다. SuperMicro의 경우에는 관련하여 DS&G에 문의하여야 한다.
