Form

minx∈Rnf(x),forf:Rn→R

where f is continuosly differentiable.

Definition : Level set

Let f:Rn→R. For given α∈R, we say the set Lα⊂Rn defined by

Lα={x∈Rn|f(x)≤α}

is called “a level set of f“

Properties

  1. If α1>α2 then Lα1⊃Lα2
  2. If α^=minf(x), then Lα^ is the solution set
  3. An algorithm solving minf(x) is called a descent method.

If the seq {xi}∞i=1 generated by algorithm, satisfy that

f(xi+1)<f(xi)∀i∈N

Definition : Tangent Set

The level set of tangent to the boundary of Lα, i.e. ∂Lα, at a point x^∈∂Lα is defined by.

T(x^,∂Lα)≜{y∈Rn|βy=limxi−x^∥xi−x^∥,xi∈β>0∂Lα∀i}

아래 그림과 같은 불연속의 경우 T(x^,∂Lα)는 아래 위로 두 원소를 가진다.

Definition : Normal Set

The set of normals to the boundary of Lα at a point x^∈∂Lα is defined by

N(x^,∂Lα)≜{ν∈Rn|⟨ν,y⟩=0,y∈T(x^,∂Lα)}

We say that ν is an outward normal at x^∈∂Lα
If f(x^+λν)>f(x^) for sufficiently small λ>0.

Theorem U-1 : Mean Value Theorem

For f∈C′,f:Rn→R

f(x)=f(y),∃s∈[0,1]such thatf(y)−f(x)=⟨∇f(x+s(y−x)),y−x⟩

(f(x)=f(y) 일 경우는 밑의 그림과 같다.)

Property U-1

Suppse that x∈Rn such that f(x)=α
If a vector ∇f(x)≠0, then ∇f(x) is outward normal to the boundary ∂Lα at x.

증명을 위해 필요한 부분

  1. ∇f(x)∈N(x,∂Lα)
  2. f(x+λ∇f(x))>f(x) for some sufficient small λ≥0

Proof of Property U-1

Proof of 1

Pick any sequence {Δxi}∞i=1 such that x+Δxi∈∂Lα and Δxi→0,∀i∈N (i.e. f(x+Δxi)=α ) <- 수식을 해석해 보면 딱 이 의미임.

For each i∈N by Mean-Value Theorem,

∃si∈[0,1]such thatf(x+Δxi)=f(x)+⟨∇f(x+siΔxi),Δxi⟩

implies that

⟨∇f(x+siΔxi),Δxi⟩=0,i∈N⇒⟨∇f(x+siΔxi),Δxi∥Δxi∥⟩=0

Let yi=Δxi∥Δxi∥, then ∃y such that yi→y.
(It means that ∥yi∥=1⇒Bo(0,1): Closed and Bounded. Compact!!)

0=limi→∞⟨∇f(x+siΔxi),Δxi∥Δxi∥⟩=⟨∇f(x+siΔxi),limi→∞Δxi∥Δxi∥⟩=⟨∇f(x+siΔxi),limi→∞yi⟩=⟨∇f(x+siΔxi),y⟩

By the definition of T(x,∂Lα)

y∈T(x,∂Lα)

Therefore, ∇f(x)∈N(x,∂Lα) (Tangent Space에 있는 y와 Inner Product 값이 0 이므로)

Proof of 2

By Mean Value Theorem

∀λ>0,∃Sλ∈[0,1]such thatf(x+λ∇f(x))=f(x)+⟨∇f(x+sλλ∇f(x),λ∇f(x)⟩)

(x+λ∇f(x)−x=λ∇f(x) … 두번쨰 항의 ⟨,⟩ 내부가 이렇게 된 이유.)

Object : f(x+λ∇f(x))−f(x)>0 for sufficiently small λ>0 implies that λ⟨∇f(x+sλλf(x)),∇f(x)⟩>0 for sufficiently small λ>0.

By Continuity of ∇f(x) at x, (즉, 어떤 ∥x∥<δ 존재하여 ∥∇f(x)∥<ε∀ε>0)

∃λ¯such that0<λ≤λ¯such that⟨∇f(x+sλλ¯∇f(x)),∇f(x)⟩≥12∥∇f(x)∥

따라서, ∇f(x) is an outward normal.

Q.E.D. of Property U-1

Corollary U-1 : descent direction

If x∈Rn is such that ∇f(x)≠0, then any vector h∈Rn such that ⟨∇f(x),h⟩<0 is a descent direction for f(⋅) at x.
i.e.

∃λ^>0such thatf(x+λ^h)−f(x)<0

Note

Differentiable Convex 의 정의에 따르면

f(y)−f(x)≥⟨∇f(x),(y−x)⟩

이다. 그러므로 y=x+λ^h 인데, ∥y∥>∥x∥ 이라면, 오히려 f(x+λ^h)−f(x)>0 이어야 한다. 이는 이미 Property U-1에서 증명된 것이다. 따라서 이것이 반대가 되기 위해서는 ∥y∥≤∥x∥ 이어야 한다, 그 경우 ⟨∇f(x+sλλ^∇f(x)),∇f(x)⟩≥12∥∇f(x)∥ 이지만, ∥y∥≤∥x∥ 이므로 부호가 바뀌어서

f(x+λ^h)−f(x)=λ^⟨∇f(x+sλλ¯∇f(x)),∇f(x)⟩≤−λ^12∥∇f(x)∥

가 된다.

증명은 생략한다.

Directional Derivative

Directional Derivative at x along the vector h is defined by

df(x,h)=limλ↓0f(x+λh)−f(x)λ

where λ↓0 means that λ is Monotone decreasing on R+, and λ↑0 is Monotone increasing on R−.

  • If f is continuosly differentiable, then

f(x+λh)−f(x)=∫10⟨∇f(x+sλh),λh⟩

imnplies that

f(x+λh)−f(x)λ=∫10⟨∇f(x+sλh),h⟩ds

Let λ↓0, then

df(x,h)=limλ↓0f(x+λh)−f(x)λ=limλ↓0∫10⟨∇f(x+sλh),h⟩ds=∫10⟨limλ↓0∇f(x+sλh),h⟩ds=⟨∇f(x),h⟩∫10ds=⟨∇f(x),h⟩

즉

df(x,h)=⟨∇f(x),h⟩

Note

많은 경우 x를 생략하고 df(h) 를 많이 사용한다. 여기서는 최적화 알고리즘 도출을 위해서 x를 굳이 표시하였다. 그 외에 h[f]=h(f) 역시 같은 의미로 사용한다. (미분 기하학, 리만 기하학, 미분 다양체론 등에서..)
