Short Course of Baysian Inference

Baysian Inference의 용어

p(θ|D)=p(D|θ)p(θ)p(D)(1)

  • A Priori (the prior) probability p(θ)
  • Likelihood p(D|θ)
  • A Posteriori (the posterior) probability p(θ|D)

Comment

  • D 로 표현한 것은 Data Dt 로 이루어진 σ-Field 라는 의미이다. 즉, 입력 데이터들은 D adapted stochastic process Dt로 볼 수 있다.
  • 따라서 입력 데이터를 서로 구별하기 위하여 index t를 도입한다면 σ-Field Dt 로 표현하여야 옳다.
  • 이렇게 표현하면, 식(1)의 경우는 좀 더 실제 알고리즘 구현에 알맞는 다음과 같은 형식으로 표현해 줄 수 있다. 먼저

    p(Dt)p(θ,Dt,Dt−1)p(θ,Dt−1)=p(Dt,Dt−1)=p(Dt|Dt−1)p(Dt−1)=p(θ,Dt)p(θ)=p(Dt|θ)

    그러므로

    p(θt|Dt,Dt−1)=p(θt,Dt,Dt−1)1p(Dt,Dt−1)=p(θt,Dt,Dt−1)p(Dt|Dt−1)p(Dt−1)=p(θt,Dt,Dt−1)p(Dt|Dt−1)p(Dt−1)p(θt,Dt−1)p(θt,Dt−1)=p(θt,Dt,Dt−1)p(θt,Dt−1)⋅1p(Dt|Dt−1)⋅p(θt,Dt−1)p(Dt−1)

    정리하면

    p(θt|Dt)=p(θt|Dt,Dt−1)=p(Dt|θt)p(θt|Dt−1)p(Dt|Dt−1)(2)

  • A Priori (the prior) probability p(θt|Dt−1), 바로 직전의 데이터 Set (Dt−1)으로 학습시킨 신경망 θt의 성능
  • Likelihood p(Dt|θt) : 신경망 θt이 데이터 Dt를 잘 만들어 낼 수 있는 가에 대한 Likelihood
  • Posteriori (the posterior) probability p(θt|Dt) : 현재까지의 데이터 Set Dt−1 으로 신경망 θt 학습

xt+1yt+1=Fxt+Gut+wt=Hxt+1+vt

yτ∈R 시스템의 출력 값 H∈R1×n 출력 전달 함수 (State -> 출력)

  • vt∈R State에 인가되는 잡음, i.i.d White Noise. Variance 는 EvtvTt=Rt, zero mean

Filtering 과정

Filtering 과정은 다음의 두 Stage를 거친다.

  • A Priori (Prediction) 과정
    • Filter Parameter (신경망의 Weight 같은 존재) θt−1|t−1
    • 입력 ut−1을 가지고
    • Prediction 데이터 θt|t−1을 만든다.
  • Posteriori (Update) 과정
    • Prediction 데이터 θt|t−1와
    • Kalman Gain(신경망에서 학습계수) 를 가지고
    • Filter Parameter를 업데이트 한다.
    • θt−1|t−1→θt|t

최적화 해야 하는 함수

원하는 목적은 이것이다.

  • xt와 θt|t 가 같았으면 좋겠다.
    • 그런데, xt와 θt|t은 둘다 Random Variable
    • 따라서, Randome Variable의 거리, 혹은 Metric인 Covariance를 목적함수로 놓는다.

      Pt|t=E(xt−θt|t)(xt−θt|t)T=cov(xt−θt|t)

  • 그래서 이것을 최소화 시키는 Kalman Gain을 찾는다. (신경망에서 Gradient Descent 와 유사하게…)

    Kt=argminKtcov(xt−θt|t)⇒∂trPt|t∂Kt=0

    • 그리고, Minimum Variancve는 θt|t가 xt의 평균 값일 떄 얻어지므로 (Dt 대신 Yt 사용, p(Xt=xt)를 p(xt)로 표시)

      p(xt|Yt)p(xt|Yt−1)=1Zexp(−12(xt−θt|t)TP−1t|t(xt−θt|t))=1Zexp(−12(xt−θt|t−1)TP−1t|t−1(xt−θt|t−1))

특별히 만일, 제어 입력 G=0 이라고 가정하면, Filter의 Baysian은 다음과 같이 명확하다.

p(Xt|Yt)=p(Xt|Yt,Yt−1)=p(Yt|Xt)p(Xt|Yt−1)p(Yt|Yt−1)

  • A priori (the prior) probability (Prediction) p(Xt|Yt−1)
  • posteriori (the posterior) probablity (Update)p(Xt|Yt)
  • Likelihood p(Yt|Xt)
왜 이렇게 표현하는가?

Estimator의 종류

  • Minimum Mean Square Error Estimation : X^MMSEt=E(Xt|Yt)=∫txt⋅p(xt|Yt)dxt
  • Maximum a Posteriori Estimation (MAP): X^MAPt=argmaxxtp(xt|Yt)
  • Maximum Likelihood (ML) Estimator : ∂fYt|Xt∂x|x=θt=0, f는 logp(Yt|Xt) 등을 사용하여 보통, 다음 함수를 최소화 시킨다.

    J=12(yt−Hθt)TR−1t(yt−Hθt)

  • Minimum Conditional KL Divergence, Minimum Conditional Accuracy Estimator등이 있다.
  • Kalman Filter는 결국 평균을 구하기는 하지만, 기본적으로 Variance를 최소화 시켜 확률을 극대화 시키는 것이므로 MAP Estimator 이다.
    • 여기서 X^MMSEt=X^MAPt=θt|t
    • 평균 개념을 사용하거나 Log-Likelihood 개념을 사용하여 업데이트 Rule을 유도한다.

Neural Network은 왜 이렇게 표현하지 못하는가?

  • 목적함수가 Estimator와 다르기 때문이다.
    • 그렇다고 못하는 것은 아니다.
    • Filter 알고리즘과 유사한 Competitive Learning 계열은 동일하게 사용 가능하다.
    • 기본적으로 Hyper Plane을 직접 기술하는 방식에서는 직접 적용이 어려울 수 있다.

  • Estimator 는 Covariance를 비교적 쉽게 정의할 수 있는 반면, Neural Network은 목적함수가 다종 다양하다.

    • Supervised Learning의 경우 (Desired Value가 있는 경우) : 일반적인 Covariance로 정의하는 대신, P Momentum Distance를 정의하고 P를 업데이트 하는 방식은 가능하다.
      • 이 경우 P가 Covariance를 대신할 수 있다.
    • 만일, Random Variable이 되도록, Update 자체에 White Noise 항을 정의하게 되면, 어떠한 목적함수를 사용하더라도 MAP 방식의 Learning이 가능하다. (연구과제)

  • 평균 개념대신 직접적으로 Weight를 업데이트 할 수 있는 기법이 있다. (가장 큰 이유)

    • Markov Chain Monte-Carlo 등.
    • 실제 예가 Boltzmann Machine

Baysian 기반 Neural Network

ANN(Artificial Neural Network)에서는 오랫동안 MLE 방식의 알고리즘을 사용해왔다.

  • A priori 확률을 알고 있는 것이 없기 떄문, 오직 데이터의 Measurement만 알고 있기 때문

    θt+1=θt−ηt∇θJ(yt,θt)=θt+ηtHTR−1t(yt−Hθt)

    • 위 알고리즘은 다음의 Measurement 확률을 최대화 시킨다. (Yt가 Gaussian, i.i.d White Noise에 의해 만들어진 yt에 의해 생성된 σ-algebra)

      p(Yt|Xt)=1Zexp(−12(yt−Hθt)TR−1t(yt−Hθt))

    • Baysian 관점에서 보면 다음과 같다. (Since we don’t know correctly a priori probability p(Xt|Yt−1) and p(Yt|Yt−1) is a uniform or meanigless probabblity such as normal factor, )

      θt=maxxt∫xxtp(Xt|Yt)dxt=maxxt∫xxtp(Yt|Xt)p(Xt|Yt−1)p(Yt|Yt−1)dxt=∫xxtmaxxtp(Yt|Xt)dP(Xt|Yt−1)dP(Yt|Yt−1)=∫xxtmaxxtf(Xt)dxt

Idea

  • A priori 확률을 정의한다.
    • Normalization Factor 로 p(Yt|Dt−1) 을 간주하자.
    • 그러면 자연스럽게 MAP Estimator로서 Baysian Neural Network이 정의된다.
  • State xt를 정의하여 알고리즘을 만들 것인가, 아니면 Weight 자체를 직접 Update 할 것인가.
    • State xt를 정의하게 되면 미분 기반의 알고리즘을 만들 수는 있다. (예 : Hopfield Network)
    • Monte Carlo 계열의 알고리즘으로 직접 Weight를 Update 하면 굳이 미분을 쓰지 않아도 된다. (예: Boltzmann Machine)

Definition of A priori Probablity

A priori Probability p(w)는 Boltzman 분포를 따른다고 했으므로 온도계수 T 에 대하여

p(w)=1Zwexp(−EwTw)=1Zwexp(−α⋅Ew)(4)

여기서 Zw는 Normalization Factor로 Zw=∫Rnexp(−α⋅Ew)dw.

  • 다시말해, w가 가질 수 있는 모든 경우에 대하여 나누어 주어야 한다.
  • 실제로 Zw를 구하여 p(w)를 구한다는 것은 경우의 수가 너무 많기 때문에 어려운 일이지만 다음과 같이 구하는 것이 일반적이다.
    • 그래서 많은 경우 Gaussian 분포를 가정하거나 다음 식을 사용하여 정규화 해야 한다.

      ∫∞−∞exp(−x2)dx=π√

  • 그러나, Baysian을 더 분석하면 Normalization Factor가 사라지거나 대치 될 수 있으므로 일단 개념적으로 놓아둔다.
Construction of Likelihood

Likelihood는 실제로 최적화 하려는 Cost를 Boltzman 분포의 Energy로 놓는다. Neural Network의 출력을 입력 x와 weight에 의해 y(x;w) 라 놓고 n번쨰 Training Data를 tn 이라 놓자. 그러면 Likelihood는 다음과 같다.

EDp(D|w)=12∑n=1N(y(x;w)−tn)2=1ZDexp(−EDTD)=1ZDexp(−βED)=∏n=1Np(tn|xn,w)(5)

where ZD=∫exp(−βED)dw

Posterior over the weights

Since the equation (3), (4), (5) are hold,

p(w|D)=p(D|w)p(w)p(D)=1ZDexp(−βED)⋅1Zwexp(−αEw)⋅1∫exp(−βED−αEw)ZDZwdw=exp(−βED−αEw)∫exp(−βED−αEw)dw=1ZSexp(−S(w))

  • 문제를 살펴보면 결국 다음 목적함수를 최적화 시키는 문제로 귀결된다.

    S(w)=12(β∑n=1N(y(x;w)−tn)2+α∥w∥2)=β2(∑n=1N(y(x;w)−tn)2+αβ∥w∥2)

  • 이때, αβ=λ 로 놓으면 1계 제한 조건을 가지는 Largrangian 문제로 변형된다.

    S(w)=β2(∑n=1N(y(x;w)−tn)2+λ∥w∥2)

  • Khun-Tucker Condition에 의해 다음이 S(w)를 위한 하나의 minimizer 필요조건이다.

    0=∇wED+λ∇wEw,λ=−∇wETw∇wED∇wETw∇wEw

Output from Baysian Neural Network.

Baysian Neural Network을 출력은 두 가지 방식이 모두 가능하다.

  • Weight Matrix로 생각하고 Fermi-Dirac 통계를 통해 출력을 내는 방법
  • Weight Vector로 생각하고 정통적인 MAP Estimation으로 출력을 내는 방법 (Radial Basis Function 방법)

Distribution over outputs

신경망 출력의 분포함수는 어떠할 지 생각해 본다.

p(y|x,D)=∫p(y|x,w)p(w|D)dw

  • 이때,, Training 데이터에 의해 MAP로 추정된 Weight, w^가 존재하고 그로 인한 신경망 출력이 y^=y(w^,x) 라고 하면, 임의의 데이터 x 가 들어왔을 때, 출력 p(y|x,D)을 Gaussian 분포 확률로 나타내고자 한다.

    p(y|x,D)≈12π−−√σyexp(−(y−y^)22σ2y)

여기서 Variance는 다음과 같다. 먼저, Gaussian 분포의 Baysian 특성은 다음과 같다.

p(x)p(y|x)p(y)=N(x,μ,Λ−1)=N(y,Ax+b,L−1)=N(y,Aμ+b,L−1+AΛ−1AT)

이므로 p(w|D)를 Gaussian 분포 q(w|D) 로 생각하면 p(y|x,D)=p(y|x,w)q(w|D) 로 볼 수 있다. 이때, q(w|D)의 mean은 w^, Variance σ2=Λ−1는 S(w)의 w에 대한 Hessian A=αI+βH 이다. 여기서 y(w,x)=y^+∇Twy(w,x)|w=w^(w−w^) 로 근사화 하면 p(y|x,w)≈N(y,y^+∇Twy(w,x)|w=w^(w−w^),(β=∂2S∂y2)−1) 그러므로 p(y|x,D)=N(y,y^,σ2) .
